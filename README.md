# Mini-AI-chatbot
AI agent using Google ADK and Gemini that acts as a personal assistant with custom tool integration.

## 📌 Project Overview
This project is a simple LLM-powered AI assistant built using **Google ADK (Agent Development Kit)** and **Gemini model**.

It demonstrates:

- LLM agent creation
- Tool integration
- Function calling
- AI assistant instruction design
- Basic agent architecture

---

##  Features
- Uses Google ADK framework
- Gemini LLM integration
- Modular agent design
- Beginner-friendly AI project structure

---

##  Technologies Used
- Python
- Google ADK
- Gemini API
- Function Tools
- Prompt Engineering
